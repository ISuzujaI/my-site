✓ Система онлайн-записи

✓ Личный кабинет  
✓ Панель администратора  

## 🎯 Как использовать

### 1. Навигация

Используйте меню в шапке сайта для перехода между страницами:

- **Sākums** / Главная - главная страница с обзором
- **Par mums** / О нас - информация о клинике
- **Pakalpojumi** / Услуги - каталог услуг с фильтрацией
- **Ārsti** / Врачи - команда врачей
- **Cenas** / Цены - прайс-лист
- **Pieraksts** / Запись - система онлайн-записи
- **Kontakti** / Контакты - контактная информация

### 2. Смена языка

В правом верхнем углу выберите язык: **LV** / **RU** / **EN**

### 3. Тестирование системы записи

1. Перейдите на страницу **Pieraksts** (Запись)
2. Выберите услугу → врача → дату → время
3. Система предложит войти или зарегистрироваться
4. После авторизации запись будет подтверждена

### 4. Тестовые аккаунты

**Администратор:**
- Email: `admin@vet.lv`
- Password: любой

**Новый пользователь:**
- Зарегистрируйтесь через форму Sign Up
- Укажите любой email (не требуется подтверждение)

### 5. Панель администратора

После входа как администратор (`admin@vet.lv`):
1. Перейдите в профиль
2. Увидите ссылку на **Administrācijas panelis**
3. Управляйте врачами, услугами, расписанием и записями

## 🔧 Backend API

Backend уже развернут и работает. Доступные endpoints:

```
GET  /doctors              - Список врачей
GET  /services             - Список услуг
POST /appointments         - Создать запись (auth)
GET  /appointments         - Мои записи (auth)
POST /signup               - Регистрация
POST /contact              - Форма обратной связи
```

## 🎨 Основные функции

### Система онлайн-записи (4 шага):
1. Выбор услуги из 6 категорий
2. Выбор врача из 6 специалистов
3. Выбор даты (следующие 14 дней)
4. Выбор времени (09:00-18:00, слоты по 30 мин)

### Личный кабинет:
- Предстоящие записи
- История посещений
- Отмена записи
- Информация профиля

### Панель администратора:
- Управление врачами (CRUD)
- Управление услугами (CRUD)
- Управление расписанием
- Просмотр всех записей
