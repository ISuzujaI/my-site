## 🎯 Первые шаги

### 1️⃣ Войти в админ-панель

**Важно! Сначала инициализируйте базу данных:**

Откройте консоль браузера (`F12`) и выполните один раз:
```javascript
fetch('https://echadzchfmifjmvyxupg.supabase.co/functions/v1/make-server-de695671/seed', {
  method: 'POST'
}).then(r => r.json()).then(console.log);
```

**Затем войдите в админ-панель:**

1. Нажмите кнопку **"Войти"** в правом верхнем углу
2. Введите данные:
   - Email: `admin@vet.lv`
   - Пароль: любой (например `12345`)
3. После входа откройте **Профиль** → **Администрационный панель**

🎉 **Готово!** Теперь вы можете:
- ➕ Добавлять врачей
- ✏️ Редактировать информацию о врачах
- 🗑️ Удалять врачей
- 👁️ Управлять статусом (активен/неактивен)

📖 **Подробная инструкция:** [ADMIN_QUICK_START.md](ADMIN_QUICK_START.md)

## 🆘 Нужна помощь?

### Частые вопросы:

**❓ Как изменить телефон в шапке?**
→ Файл: `/src/app/utils/constants.ts`, строка 11-14

**❓ Как добавить нового врача?**
→ Файл: `/supabase/functions/server/seed-data.ts`, массив `seedDoctors`

**❓ Как изменить цвет кнопок?**
→ Файл: `/src/styles/theme.css`, переменная `--color-green`

**❓ Текст не меняется на всех языках?**
→ Нужно изменить во всех трех секциях (`lv`, `ru`, `en`) в `LanguageContext.tsx`

**❓ Изображение не загружается?**
→ Проверьте, что URL начинается с `https://` и изображение доступно публично
