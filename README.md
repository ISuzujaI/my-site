# 🏥 SIA Abuvet - Сайт ветеринарной клиники

Современный многостраничный веб-сайт ветеринарной клиники с поддержкой трех языков (латышский основной, русский, английский) и адаптивным дизайном.

## 🎯 Функционал

- 🌍 Поддержка 3 языков (LV, RU, EN)
- 📱 Адаптивный дизайн
- 📅 Система онлайн-записи к врачу
- 👨‍⚕️ Панель администратора
- 🔒 Авторизация пользователей
- 💾 Интеграция с Supabase
- 🎨 Фирменный стиль клиники

## 🚀 Быстрый старт

```bash
npm install
npm run dev
```

## 📦 Развертывание

```bash
# Сборка фронтенда
npm run build

# Деплой на Supabase
npm run deploy:supabase

# Деплой на Vercel/Netlify
npm run deploy:vercel
npm run deploy:netlify
```
