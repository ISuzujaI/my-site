# VetClinic - Ветеринарная клиника

Современный, многостраничный веб-сайт ветеринарной клиники с системой онлайн-записи.

## 🚀 Функции

### Основные возможности
- ✅ **Многостраничная структура** с React Router
- ✅ **Мультиязычность** (латышский, русский, английский)
- ✅ **Адаптивный дизайн** (desktop + mobile)
- ✅ **Система онлайн-записи** с выбором врача, услуги, даты и времени
- ✅ **Авторизация пользователей** (регистрация/вход)
- ✅ **Личный кабинет** с историей посещений
- ✅ **Панель администратора** для управления клиникой
- ✅ **Backend на Supabase** с REST API

### Страницы сайта

1. **Главная** (`/`)
   - Краткое описание клиники
   - Преимущества (4 блока)
   - Статистика (опыт, врачи, пациенты)
   - Превью услуг
   - Отзывы клиентов
   - CTA блок

2. **О клинике** (`/about`)
   - История клиники
   - Ценности (4 блока)
   - Современное оборудование
   - Лицензии и сертификаты
   - Галерея

3. **Услуги** (`/services`)
   - Тerapия
   - Хирургия
   - Вакцинация
   - Диагностика
   - Стоматология
   - Экстренная помощь
   - Фильтрация по типу услуги

4. **Врачи** (`/doctors`)
   - Список из 6 врачей
   - Профили с фото, специализацией, опытом
   - Возможность записи к конкретному врачу

5. **Цены** (`/prices`)
   - Таблица услуг по категориям
   - Цены в EUR
   - Важная информация

6. **Онлайн-запись** (`/booking`)
   - Пошаговый процесс (4 шага)
   - Выбор услуги
   - Выбор врача
   - Выбор даты
   - Выбор времени
   - Отображение доступных слотов в реальном времени
   - Подтверждение записи

7. **Личный кабинет** (`/profile`)
   - Информация о пользователе
   - Предстоящие записи
   - История посещений
   - Отмена/перенос записи

8. **Контакты** (`/contact`)
   - Адрес, телефон, email
   - Часы работы
   - Форма обратной связи
   - Карта (placeholder)

9. **Панель администратора** (`/admin`)
   - Управление врачами
   - Управление услугами
   - Управление расписанием
   - Управление записями

## 🔑 API Endpoints

### Авторизация
- `POST /signup` - Регистрация нового пользователя

### Врачи
- `GET /doctors` - Получить всех врачей
- `GET /doctors/:id` - Получить врача по ID
- `POST /doctors` - Создать/обновить врача (admin)

### Услуги
- `GET /services` - Получить все услуги

### Записи
- `GET /appointments` - Получить записи пользователя (auth)
- `POST /appointments` - Создать запись (auth)
- `DELETE /appointments/:id` - Отменить запись (auth)
- `GET /available-slots?doctorId=&date=` - Получить доступные слоты

### Контакты
- `POST /contact` - Отправить форму обратной связи

## 📝 Важные примечания

1. **Supabase интеграция**: Backend уже настроен, но для полной функциональности требуется подключение к Supabase.

2. **GDPR соответствие**: Для продакшн использования необходимо:
   - Добавить политику конфиденциальности
   - Настроить защиту персональных данных
   - Добавить согласие на обработку данных
   - Настроить безопасное хранение медицинских данных

3. **Email/SMS уведомления**: В текущей версии комментированы. Для активации:
   - Настроить email сервер в Supabase
   - Настроить SMS шлюз
   - Раскомментировать соответствующий код

4. **Карта**: На странице контактов используется placeholder. Для интеграции Google Maps:
   - Получить API ключ Google Maps
   - Добавить компонент карты
   - Указать координаты клиники

5. **Изображения**: Используются из Unsplash. Для продакшн версии замените на реальные фото клиники.
